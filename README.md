
# 🧭 Retorgetry - Lost & Found Website

Retorgetry is a simple and modern **Lost & Found web platform** designed for college/university students and communities to help people report and find lost items easily.

---

## 🚀 Features

- 📌 Report Lost Items
- 📌 Report Found Items
- 🔍 Search and Filter Items
- 👤 User Authentication (Login/Register)
- 🗂️ Item categorization (ID card, phone, books, etc.)
- 📩 Contact between finder and owner
- 📱 Responsive UI (Mobile + Desktop)

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, Bootstrap
- Backend: PHP
- Database: MySQL
- Others: JavaScript (optional for interactivity)

## ⚙️ Installation Guide

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/retorgetry.git
````

2. Move project to your local server (htdocs for XAMPP)

3. Import database file into MySQL

4. Configure `db.php` with your database credentials

5. Run project in browser:

```
http://localhost/retorgetry/
```
